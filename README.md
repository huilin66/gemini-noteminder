General Overview
This project was generated by Gemini 3 Pro Preview and completed with Vibe Coding.
Welcome to NoteMinder! This application is designed to help you organize tasks effectively using a skeuomorphic interface.
Bookshelf: Your primary workspace background. It holds interactive links and provides a clean desktop environment.
Notebooks: The core data structure. Create multiple notebooks (e.g., Work, Personal) to categorize your tasks.
Note: A sticky-note view of your tasks. Pin tasks from your notebook to your desktop to keep them visible and editable.
